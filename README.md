
# NextJS Portfolio Website

Simple Portfolio website with below sections : 

- Header
- Hero section
- Projects
- Technologies
- Timeline 
- Accomplishments
- Footer

## Run Locally

Clone the project

```bash
  git clone https://github.com/nlagdhir/nextjs-portfolio.git
```

Go to the project directory

```bash
  cd nextjs-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
